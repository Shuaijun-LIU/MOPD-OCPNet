Framework Features

Multi-Source Ocean Data

Bathymetry (GEBCO 2025), ocean physics (CMEMS GOPAF), biodiversity (GMB), and microplastics (NOAA-NCEI)

Marine Robotics Simulation

Configurable 3-DOF and 6-DOF systems with APIs for custom robot design and hierarchical control

PollutionModel3D

Mechanistic simulation engine integrating physical, chemical, and biological processes for pollutant dispersion

Modular Architecture

Scalable framework supporting custom training tasks and seamless data-simulation integration
