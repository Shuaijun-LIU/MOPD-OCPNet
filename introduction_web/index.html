Dataset Features

Terrain Data

High-resolution bathymetry from GEBCO 2024 with 15 arc-second grid resolution

Ocean Currents

Comprehensive current data from Copernicus Marine Service (GOPAF)

Physical Parameters

Includes salinity, temperature, and sea surface height data

Integrated Data

Harmonized grids using RBF interpolation for seamless integration
