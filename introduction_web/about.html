About the Project

Learn about our mission, team, and the development of OceanEnv

Our Mission

The OceanEnv project aims to provide researchers, educators, and marine robotics developers with a comprehensive, modular simulation environment by unifying large-scale ocean data with advanced simulation tools for marine robotics training.

January 2024

Project Inception

Initial planning and data source identification

March 2024

Data Collection

Started collecting and processing GEBCO and GOPAF data

July 2024

Integration Development

Development of data integration algorithms and quality control procedures

December 2024

Beta Testing

Initial release and testing with research partners

March 2025

Public Release

Official public release of the Combined Ocean Environment Dataset
