About the Project

Learn about our mission, team, and the development of the Combined Ocean Environment Dataset

Our Mission

The Combined Ocean Environment Dataset project aims to provide researchers, educators, and ocean enthusiasts with comprehensive, high-quality ocean data by integrating multiple data sources into a unified, easy-to-use format.

January 2023

Project Inception

Initial planning and data source identification

March 2023

Data Collection

Started collecting and processing GEBCO and GCPAF data

July 2023

Integration Development

Development of data integration algorithms and quality control procedures

December 2023

Beta Testing

Initial release and testing with research partners

March 2024

Public Release

Official public release of the Combined Ocean Environment Dataset
